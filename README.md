#  EMI Calculator

A modern and user-friendly EMI (Equated Monthly Installment) Calculator built using HTML, CSS, and JavaScript.  
It helps users calculate monthly loan payments along with total interest and total payment.

---

## 🚀 Features

- 📊 Calculate Monthly EMI  
- 💵 Displays Total Interest Payable  
- 💰 Shows Total Payment (Principal + Interest)  
- 🎯 Simple and clean UI  
- ⚡ Fast and responsive  

---

## 🛠 Tech Stack

- HTML  
- CSS  
- JavaScript
